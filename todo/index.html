    In source code, comments # Info 12 refer to the following procedures:

    1. FTG started, (re)loaded preferences
    2. Plane position
    3. Airport name
    4. Loading loading airport data
    5. Airport has ATC ground
    6. Airport has routing network (# nodes, # edges)
    7. Airport runways
    8. Airport ramps/stands
    9. Airport loaded
    10. Departure or arrival (guess, changeable)
    11. Selected destination.
    12. Route found.
    13. Lights, segments, stop bars placed; first segment lit.
    14. Started.
    15. Next segment.
    16. Terminated successfully. (One more happy pilot.)
    17. Cancelled by pilot.
